How to Run it
----------------


1. Open a terminal change directory to program's directory In Windows cd \d "folder/path"

2. Must install GCC compiler 

3. type - make

4. Input what ever u want. NOTE: Remember Don't put more character than 1000  // Max limit 1000

5. put EOF end of the line in windows 'CTRL + Z' press enter


IMPORTANT: Change stdio.h header file path on c_cpp_properties.json
